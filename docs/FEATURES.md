# Features

A detailed catalog of what makes AveGram, AveGram.

## 🎨 Design

- **Modern Glass UI** — translucent surfaces, soft shadows, generous spacing
- **Deep Dark Mode** — true `#0D1117` background, frosted card overlays
- **Material You** — dynamic color on Android 12+
- **Custom launcher icons** — pick from a curated set under Settings → Appearance
- **Adaptive corner radius** — 16–24 dp across cards, sheets, and dialogs

## 🔒 Privacy

- **Ghost Mode** — read messages without sending a "seen" receipt
- **Stealth typing** — disable typing indicators
- **No-recent-actions log** — silently bypass action history when needed
- **Message-log control** — fine-grained toggle for what gets recorded locally
- **Encrypted local DB** — sensitive caches stored with platform keystore

## 💬 Messaging

- **Saved deleted messages** — keep a local copy of messages deleted by sender
- **Edit history** — long-press a message → view full revision timeline
- **Double-tap actions** — configurable (react, reply, edit, save…)
- **Bookmarks** — pin messages locally without notifying the chat
- **Quick translate** — translate any message inline
- **Sticker set helpers** — quick install / remove / preview

## 🤖 AI

- **In-chat translator** — supports OpenAI, Google Gemini, DeepSeek
- **System AI service helper** — bridges to on-device AI providers (where available)
- **Configurable prompts** — tweak tone & style per language pair

## 📂 Organization

- **Custom folders** — beyond the default tabs, fully reorderable
- **Per-chat settings** — overrides for notifications, theme accent, fonts
- **Cache control** — granular per-chat / per-media-type cleanup
- **Bookmark manager** — dedicated screen for saved messages

## ⚡ Power tools

- **Settings backup** — export / import all preferences
- **Premium status helper** — local-only premium feature toggles for testing
- **Peer color helper** — customize chat header tints
- **Now Playing** — system media controls integrated in chat header

## 🌐 Translations

AveGram inherits the full Telegram localization matrix and adds AI-powered
on-the-fly translation for any message in any chat.
